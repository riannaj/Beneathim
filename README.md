# Beneath — Clothing Website

A modern clothing e-commerce site built with React and Vite.

## Setup

```bash
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
```

Output goes to the `dist` folder. Use `npm run preview` to preview the production build locally.

## Features

- **Hero** — Full-height hero with seasonal messaging
- **Product grid** — Catalog of clothing with images (Unsplash placeholders)
- **Shopping cart** — Add/remove items, update quantity, view subtotal
- **Responsive** — Mobile-friendly layout with a slide-out cart

Edit `src/data/products.js` to change products. Replace image URLs with your own assets when ready.
